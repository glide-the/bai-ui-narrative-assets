## ✨ 夜间主题办公界面UI设计方案（Dark Mode）


### 🎨 总体视觉风格  
以**「暗夜科技感+精准交互」**为核心，结合深色调的沉浸感与高频功能的清晰导向：  
- **主色调**：深空黑背景（`#0F0F12`）+ 哑光深灰组件（`#2D2D33`）+ 霓虹橙色（`#FF6B00`）作为交互强调色；  
- **布局规范**：遵循8px网格系统，模块边缘采用12°斜切角，营造锐利科技感；  
- **微质感**：输入框/卡片表面添加10%灰度噪点，hover时浮现1px内发光晕（glow）；  
- **排版层级**：标题用`Noto Sans SC 700（32px）`，正文用`Noto Sans SC 400（16px）`，辅助文字用`Noto Sans SC 300（14px）`。  


### 🧩 核心模块设计（对应原图布局）  
#### 1. 左侧导航栏（功能入口区）  
- **布局**：垂直排列5个核心功能图标（品牌标识→文件→通知→时钟→设置），图标尺寸24×24px，内边距16px；  
- **交互细节**：  
  - 通知图标（带“7”角标）：橙色圆角小方块（16×16px），数字用10px粗体，每30s轻微闪烁（透明度动画）；  
  - 图标hover效果：背景变为深灰（`#2D2D33`），圆角8px，无额外动效干扰。  


#### 2. 顶部问候区（情感连接点）  
- **内容**：居中显示`“Hey dmeck, got a burning question?”`，白色粗体，字间距-0.5px，字母间距0.8px；  
- **动态适配**：根据时间切换文案（早/中/晚分别对应“Good morning/afternoon/evening”），强化场景感。  


#### 3. 中央输入交互区（核心操作区）  
- **输入框**：深灰圆角矩形（`#2D2D33`，圆角12px），提示文字“Press i chat”用浅灰（`#8A8A94`），聚焦时边框变为橙色（2px），底部浮现3px橙色进度条；  
- **“+ Add”按钮**：左侧嵌入，白色文字，hover时背景变为`#3D3D45`（0.2s过渡）；  
- **发送箭头**：右侧橙色图标（`#FF6B00`），点击时缩放至0.9倍（0.2s bounce动画），加载时旋转180°（0.3s ease）。  


#### 4. 功能卡片区（高频任务入口）  
- **布局**：2×3网格排列6张卡片（小屏适配为1×6），每张卡片尺寸320×100px，圆角12px，边框用1px半透明白色（`rgba(255,255,255,0.2)`）；  
- **交互细节**：  
  - 卡片hover：上浮3px，边框变为橙色（1px），阴影加强（`0 8px 24px rgba(0,0,0,0.3)`）；  
  - 图标统一：右侧用绿色文档图标（`#00C48C`），区分单文档/双文档（如“Meeting Preparation”用双文档堆叠）。  


### 🔧 技术实现要点  
#### 1. 核心动效代码（CSS）  
```css
/* 输入框聚焦动画 */
.input-focus {
  transition: all 0.3s;
  border-color: #FF6B00;
  box-shadow: 0 0 0 4px rgba(255, 107, 0, 0.1);
}

/* 卡片hover上浮 */
.card-hover {
  transition: all 0.3s;
  transform: translateY(0);
}
.card-hover:hover {
  transform: translateY(-3px);
  border-color: #FF6B00;
  box-shadow: 0 8px 24px rgba(0,0,0,0.3);
}

/* 通知角标闪烁 */
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
.notification-badge {
  animation: pulse 2s infinite cubic-bezier(0.4, 0, 0.6, 1);
}
```


#### 2. 响应式适配  
- **大屏（>1200px）**：6卡片2×3布局，输入框固定宽600px；  
- **中屏（768-1200px）**：6卡片2×3布局，输入框宽80%；  
- **小屏（<768px）**：6卡片1×6布局，左侧导航折叠为底部Tab栏（图标+文字）。  


### 🎯 设计价值  
这套方案兼顾了**夜间模式的沉浸感**与**办公场景的高效性**：深色调减少视觉疲劳，橙色强调色快速引导关键操作，模块化布局让高频功能一目了然。所有动效与样式均围绕“精准交互”设计，避免过度装饰，确保用户专注于任务本身。
