## 对话历史列表设计

对话页面的消息流布局规范。

---

### 1. 消息类型与布局

| 消息类型 | 对齐 | 容器样式 | 说明 |
|---------|------|---------|------|
| 👤 用户消息 | 右 | 浅色卡片（圆角、阴影） | 支持文本 + 图片缩略图 + 附件 |
| 💬 助手文本 | 左 | 无容器，纯文本 | Markdown 渲染，直接显示在消息流中 |
| 🔧 工具步骤 | 左 | 可折叠区块 | 橙色左边线 + 斜体标题 + 折叠/展开箭头 |
| 💻 终端输出 | 左 | 深色代码块 | 顶部标题栏 "Terminal" + 复制按钮，底部 Exit code |
| 📊 会话统计 | 居中 | 统计卡片 | Token/耗时/费用 |

### 2. 各消息详细规范

#### 用户消息（右对齐）
- 浅色背景卡片，圆角 `rounded-2xl`
- 卡片内可包含：文本、图片缩略图、文件附件
- 图片缩略图小尺寸展示，文本紧跟其下
- 链接用橙色高亮

#### 助手文本（左对齐）
- 无背景、无边框，直接渲染文本
- 支持 Markdown（标题、列表、加粗等）
- 段落间正常间距

#### 工具步骤（可折叠）
- 橙色左边线（`border-l-2 border-accent`）
- 标题为斜体，描述工具操作（如 "Cloning the GitHub repository"）
- 右侧折叠箭头，点击展开/收起详情
- 默认收起，展开后显示详细执行过程

#### 终端输出（深色代码块）
- 深色背景 `bg-[#1a1a1a]`，圆角
- 顶部栏：左侧 "< Terminal" 标题，右侧复制图标
- 命令行：`$` 符号橙色，命令白色，等宽字体
- 输出文本浅灰
- 底部 Exit code：成功 `0` 绿色，失败红色

### 3. 布局结构

```
ChatPanel (flex flex-col min-h-0)
├── Header (右上角 "+ New Chat" 按钮)
├── ChatMessageList (flex-1 overflow-y-auto, max-w-3xl mx-auto)
│   ├── 👤 用户卡片 (右对齐, 浅色背景)
│   ├── 💬 助手文本 (左对齐, 无容器)
│   ├── 🔧 可折叠步骤 (左对齐, 橙色左边线)
│   ├── 💻 终端代码块 (左对齐, 深色背景)
│   ├── 💬 助手文本 (左对齐, 无容器)
│   ├── 🔧 可折叠步骤 ...
│   └── ...
└── AIInputDock (sticky bottom-0)
    ├── 输入框 (placeholder "Press i chat")
    ├── "+ Add" 按钮 (左下)
    └── 发送按钮 (右下, 橙色箭头)
```

### 4. 响应式

- **桌面**：消息流 `max-w-3xl` 居中
- **移动端（<768px）**：全宽，内边距缩小
