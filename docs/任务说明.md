# 任务说明

## 1. 项目名称

**洱海边的三重回声：白族文化与现代文明叙事型 UI 设计**

副标题：**从白族文化到现代文明的视觉叙事**

---

## 2. 项目背景

本项目基于一组民族图案符号资产，将其从静态图录重构为一个具有故事感、文化深度与现代交互逻辑的单屏叙事型 UI 系统。

设计不以“展示符号”为唯一目标，而是以白族文化与现代文明之间的关系为叙事核心：传统文化如何在古道流动、村寨生活、本主信仰、三道茶、花文化与民居建筑中形成，又如何在现代旅游、市场、数字界面、建筑更新和公共文化活动中发生变化。

白族文化具有鲜明的地域性、开放性和生活性。它与大理、洱海、古道文化、本主文化、三道茶、花文化、白族民居等元素密切相关。现代文明进入后，传统并不是简单消失，而是经历了转译、变形、再组织与再生长。

本项目要表达的核心判断是：

> 传统不必只停留在博物馆里，它可以成为现代 UI、空间、品牌、建筑、餐饮、文旅和公共文化活动的一部分。

---

## 3. 项目目标

### 3.1 设计目标

1. 将“身份符号 / 生活符号 / 仪式符号”从静态分类转化为叙事线索。
2. 构建一个单屏横向故事板 UI，避免长屏滚动。
3. 通过白族文化与现代文明的关系，呈现文化的延续、冲突、转化与共生。
4. 形成可交付给设计师、前端工程师、内容策划与项目负责人使用的资产文本。
5. 为后续 Figma 设计、前端开发、展陈屏、网页首屏或文旅交互产品提供基础规范。

### 3.2 内容目标

通过七个章节完成叙事：

1. 古道：文化从流动中来
2. 村寨：身份被看见
3. 本主：神圣走近人间
4. 三道茶：传统成为世界语言
5. 花与日常：自然进入生活
6. 民居之困：传统被现代挤压
7. 新的共生：传统成为现代界面

### 3.3 交互目标

用户不通过长屏阅读，而是在一个完整视野内完成探索：

- 点击章节进入故事节点
- 点击符号查看意义
- 点击画面区域触发局部解释
- 使用底部横向进度条切换章节
- 使用弹层查看符号详情
- 使用详情面板理解文化转喻

## 5. 核心设计问题

本项目不是问：

> 如何把民族图案摆得更好看？

而是问：

> 如何让白族文化在现代界面中重新获得叙事位置？

因此设计要处理三组关系：

### 5.1 传统与现代

传统不是复古装饰，现代也不是水泥、玻璃和数字屏幕的简单堆叠。  
设计要表现传统如何进入现代系统，现代如何改变传统表达。

### 5.2 符号与生活

符号不是孤立图案，而是生活局部对整体文化的转喻。  
头饰代指身份，陶罐代指日常，手掌代指仪式，三道茶代指待客、哲理和市场化传播。

### 5.3 文化保护与文化再生

文化保护不是把传统封存起来。  
更有效的路径是提取传统中的结构、秩序、材料、节奏和精神，使其进入新的生活场景。

## 6. 设计原则

### 6.1 单屏叙事原则

页面主体不做长屏滚动。  
所有信息通过横向章节切换、符号弹层、详情面板、底部索引和局部交互完成。

### 6.2 克制表达原则

避免廉价民俗风，避免旅游纪念品感。  
不使用过度饱和色彩，不堆砌民族纹样，不把文化变成装饰噪音。

### 6.3 叙事优先原则

每个符号必须服务章节叙事。  
符号不能只是“好看”，必须承担文化说明、情绪推进或现代转化功能。

### 6.4 文化转喻原则

符号解释采用“局部代整体”的方式：

- 头饰 → 身份 → 族群位置
- 陶罐 → 器物 → 日常生活
- 手掌 → 身体动作 → 仪式秩序
- 花叶 → 自然 → 饮食、节日与庭院
- 民居 → 空间组织 → 生活方式

### 6.5 现代容器原则

最终页面要证明传统可以进入现代容器：

- UI 图标
- 品牌系统
- 文旅导览
- 展陈屏幕
- 空间装饰
- 餐饮体验
- 民居更新
- 公共文化活动

## 9. 页面限制条件

1. 主体页面不得出现长屏纵向滚动。
2. 推荐画布尺寸为 `1440 × 900 px`。
3. 主体信息必须在一屏内完成组织。
4. 超出信息使用弹层、抽屉、Tab、横向切换或局部放大解决。
5. 符号不应全部同时堆在页面上，应按章节和叙事需要出现。
6. 不做纯图录，不做传统文化百科页。
7. 不做过度商业化的文旅宣传页。
8. 不做单纯“民族风装饰界面”。
9. 叙事与 UI 结构必须一致，不能文案讲故事、页面仍然只是符号陈列。

## 10. 设计验收标准

### 10.1 叙事验收

- 是否能清楚看出“白族文化与现代文明”的关系？
- 是否完成从古道、村寨、本主、三道茶、花、民居到现代界面的推进？
- 是否每一章都有明确的文化主题和现代转折？
- 是否避免了空泛文化口号？

### 10.2 UI 验收

- 是否坚持单屏横向故事板？
- 是否存在不必要的长屏滚动？
- 左侧旁白、右侧画面、底部章节导航是否清晰？
- 章节切换是否自然？
- 符号详情是否容易查看？
- 信息密度是否适中？

### 10.3 视觉验收

- 是否体现米纸、墨黑、陶土红、石纹、布纹、纸纹等材质感？
- 是否避免廉价民俗风？
- 是否有白族民居、洱海、古道、本主、三道茶、花文化等视觉暗示？
- 传统符号是否与现代 UI 组件融合，而不是贴图式装饰？

### 10.4 内容验收

- 每个符号是否有明确用途？
- 旁白是否短而有力？
- 是否保留了文化复杂性？
- 是否准确表现现代文明带来的转化、冲突和再生？
- 是否提供了可直接用于页面的文案资产？

---

## 11. 项目一句话说明

> 本项目将身份、生活与仪式三类民族符号，重构为一条白族文化进入现代文明的横向叙事路径：古道带来流动，村寨形成身份，本主维系共同体，三道茶走向世界，花进入日常，民居遭遇现代化冲突，最终让传统符号重新成为现代设计的界面语言。
