# 完整叙事设计资产

## 1. 总叙事主题

**洱海边的三重回声**

副标题：**从白族文化到现代文明的视觉叙事**

这是一条从古道走向现代界面的故事线。  
故事不讲一个具体人物，而讲一组符号如何从生活中生长出来，又如何在现代文明中重新找到自己的位置。

三重回声分别是：

```text
身份的回声：我们如何被看见？
生活的回声：我们如何生活？
仪式的回声：我们如何记住时间、自然与共同体？
```

---

## 2. 故事世界观

大理不是孤立的风景，而是文化流动的节点。  
古道经过这里，带来商旅、信仰、器物和风俗。  
村寨在这里形成，身份在服饰与礼俗中被看见。  
本主信仰在村头、庙堂与日常生活中维系共同体。  
三道茶从待客之礼变成世界认识白族的窗口。  
花进入庭院、饮食、节日和市场。  
民居在现代化中遭遇挤压，也逼迫人们重新思考传统如何继续存在。  
最后，传统不再只作为遗产被保存，而是成为现代界面、品牌、空间和生活方式的一部分。

故事的最终判断：

> 现代文明不是让传统退出生活，而是迫使传统重新寻找容器。

---

|---|---|---|---|---|

# 章节 01｜古道：文化从流动中来

## 1. 叙事目标

让用户理解：白族文化不是封闭形成的，而是在古道、商旅、迁徙、交流中不断吸收与再组织。

## 2. 章节标题

```text
01 古道
文化从流动中来
```

## 3. 画面构成

画面中心是一条从左向右延伸的黑色曲线，既像古道，也像洱海岸线。  
曲线不是笔直道路，而是带有手绘感的流动线。  
线上依次出现鸟、陶罐、太阳旋涡。

背景可以加入极淡的山影和水纹，但不能抢画面主体。

## 4. 使用符号

```text
B6 鸟
B7 陶罐
C3 太阳旋涡
```

## 5. 符号功能

- 鸟：迁徙、流动、自然路径
- 陶罐：器物交换、生活流通
- 太阳旋涡：时间、循环、文明记忆

## 6. 旁白文案

```text
大理不是孤立的地方。
古道经过这里，商旅、信仰、器物、语言和风俗在这里交汇。
文化不是静止的遗产，而是一次次经过、停留、交换之后留下的生活纹路。
```

## 7. 现代文明转折

```text
今天的高速路、机场、手机地图和文旅路线，延续了古道的功能。
只是流动的形式变了：过去是马帮、器物和口述传说，今天是游客、影像、品牌和数字信息。
```

## 8. 交互方式

- 点击古道线：线条缓慢亮起，三个符号依次出现。
- Hover 鸟：显示“迁徙与交流”。
- Hover 陶罐：显示“器物带来生活方式”。
- Hover 太阳旋涡：显示“时间留下文明纹路”。

## 9. 情绪关键词

```text
流动 / 开放 / 远方 / 起源 / 时间
```

## 10. 本章短句资产

```text
流动，让文化形成。
古道不是路，是文化经过的痕迹。
每一次经过，都会留下新的生活方式。
```

# 章节 02｜村寨：身份被看见

## 1. 叙事目标

表现身份如何通过服饰、头饰、耳饰、项圈等身体装饰被共同体识别。

## 2. 章节标题

```text
02 村寨
身份被看见
```

## 3. 画面构成

右侧出现抽象村寨轮廓：白墙、青瓦、照壁、院落。  
中央不是完整人物，而是由头饰、耳饰、项圈、冠饰组成一个“身份拼贴人物”。

人物轮廓可以半透明，符号作为可见部分逐个亮起。

## 4. 使用符号

```text
A1 羽冠 / 头饰
A3 圆形耳饰
A5 项圈 / 颈饰
A8 冠饰
```

辅助符号：

```text
A4 垂坠腰饰 / 服饰边
A6 弧形饰带
```

## 5. 符号功能

- 头饰：身份入口
- 耳饰：身体装饰与族群审美
- 项圈：身体边界
- 冠饰：节庆与仪式身份
- 饰带：服饰结构与身体秩序

## 6. 旁白文案

```text
身份首先不是一句介绍，而是被看见的方式。
头饰、耳饰、项圈、衣边，让一个人从普通身体进入族群秩序。
在村寨里，身份不是孤立的个人标签，而是人与家庭、节庆、礼俗之间的关系。
```

## 7. 现代文明转折

```text
今天，身份被更多界面重新编码。
头像、昵称、证件、品牌、社交媒体和城市职业，构成新的身份系统。
但在白族文化里，身体装饰曾经就是最直接的界面。
```

## 8. 交互方式

- 点击人物轮廓：身份符号逐个拼合。
- 点击 A1：弹出“头饰是身份最先被看见的地方”。
- 点击 A3：弹出“耳饰让身体进入装饰秩序”。
- 点击 A5：弹出“项圈标记身体边界与礼俗位置”。

## 9. 情绪关键词

```text
可见 / 归属 / 身体 / 礼俗 / 识别
```

## 10. 本章短句资产

```text
服饰不是装饰，是身份的边界。
被看见，是进入共同体的第一步。
一个头饰，也是一种身份秩序。
```

# 章节 03｜本主：神圣走近人间

## 1. 叙事目标

表现本主文化如何维系人与共同体、人与自然、人与时间之间的关系，并呈现其从祭祀到公共文化活动的现代转化。

## 2. 章节标题

```text
03 本主
神圣走近人间
```

## 3. 画面构成

背景为抽象本主庙门楼。  
中央不出现具体神像，而用手掌、牛角兽骨、太阳旋涡、象头图腾和双人舞构成仪式核心。

画面左半部分更传统：庙门、供桌、纹样。  
画面右半部分加入现代元素：舞台灯、扩音器、手机拍摄、村民聚会。

## 4. 使用符号

```text
C1 张开手掌
C2 牛角兽骨
C3 太阳旋涡
C4 象头图腾
C6 双人舞
```

## 5. 符号功能

- 张开手掌：身体参与仪式
- 牛角兽骨：古老记忆、祖灵、生命循环
- 太阳旋涡：神圣中心、时间循环
- 象头图腾：力量、守护、图腾性
- 双人舞：共同体欢庆与身体秩序

## 6. 旁白文案

```text
本主不是遥远的神。
它在村头，在庙里，也在人们的日常判断里。
白族本主文化中，人与本主、人与自然、人与人之间并不是绝对隔开的关系，而是一种平等、宽容、共处的秩序。
```

## 7. 现代文明转折

```text
当现代生活进入村寨，仪式没有消失。
它改变了功能：从单纯祭祀，转向聚会、表演、交流和共同体记忆。
娱神仍在，但娱人变得更加明确。
```

## 8. 交互方式

- 点击庙门：仪式核心符号出现。
- 点击双人舞：切换到公共文化活动画面。
- 点击“现代转折”：舞台、灯光、手机拍摄淡入。
- Hover 手掌：显示“身体参与文化秩序”。

## 9. 情绪关键词

```text
庄重 / 亲近 / 共处 / 共同体 / 神圣日常化
```

## 10. 本章短句资产

```text
神圣不在远方，它在村头。
仪式没有消失，只是换了功能。
本主维系的不是恐惧，而是共同体的关系。
```

# 章节 04｜三道茶：传统成为世界语言

## 1. 叙事目标

表现三道茶如何从白族待客习俗转化为现代文旅品牌与世界理解白族文化的窗口。

## 2. 章节标题

```text
04 三道茶
传统成为世界语言
```

## 3. 画面构成

三只茶杯横向排列。  
每只茶杯对应一个阶段：

```text
一苦
二甜
三回味
```

茶杯上方有轻微蒸汽，蒸汽可变成花叶、太阳旋涡和古道线。

背景加入堂屋、茶桌、民居窗格的抽象轮廓。

## 4. 使用符号

```text
B7 陶罐
B4 花叶枝蔓
C3 太阳旋涡
```

辅助符号：

```text
B3 笛 / 长杆乐器
A6 弧形饰带
```

## 5. 符号功能

- 陶罐：烤茶、器物、待客
- 花叶：香气、自然、甜味
- 太阳旋涡：回味、哲理、时间沉淀
- 饰带：仪式化呈现

## 6. 旁白文案

```text
三道茶把生活经验压缩成一次饮茶过程。
一苦，二甜，三回味。
它既是待客之礼，也是白族文化进入现代旅游、市场和世界交流的窗口。
```

## 7. 三杯茶文案

```text
一苦：生活的起点
二甜：人与人的款待
三回味：经验沉淀成智慧
```

## 8. 现代文明转折

```text
当三道茶进入旅游、展演和品牌包装，它不再只是村寨堂屋里的待客之礼。
它成为一种可传播、可体验、可记忆的文化界面。
问题不在于市场化是否发生，而在于市场化之后，茶是否仍保留生活的温度。
```

## 9. 交互方式

- 点击第一杯：苦茶说明浮现。
- 点击第二杯：甜茶说明浮现。
- 点击第三杯：回味茶说明浮现。
- 点击“走向世界”：画面出现菜单、游客、包装、线上预约图标。

## 10. 情绪关键词

```text
款待 / 哲理 / 品味 / 传播 / 温度
```

## 11. 本章短句资产

```text
一苦，二甜，三回味。
茶是味道，也是人生秩序。
传统成为世界语言时，不能失去生活的温度。
```

# 章节 06｜民居之困：传统被现代挤压

## 1. 叙事目标

表现白族传统民居在现代化中的两难处境：传统建筑样式、材料、空间组织和生活方式正在被现代建筑取代。

## 2. 章节标题

```text
06 民居之困
传统被现代挤压
```

## 3. 画面构成

画面左右对比。

左侧：

```text
白墙
青瓦
照壁
石墙
天井
庭院花木
```

右侧：

```text
水泥楼房
玻璃窗
广告牌
电线
方盒子建筑
```

中央是一条可拖动滑块。  
滑块左侧是传统民居，右侧是现代楼房。

## 4. 使用符号

```text
A4 垂坠腰饰 / 服饰边
A7 弧形腰饰
B7 陶罐
C7 牛角颅骨
```

辅助符号：

```text
B4 花叶枝蔓
C3 太阳旋涡
```

## 5. 符号功能

- 饰带：传统装饰结构
- 陶罐：日常生活遗存
- 牛角颅骨：被压缩的仪式记忆
- 花叶：庭院生活
- 太阳旋涡：传统时间感

这些符号不再完整展开，而是以“碎片”的形式散落在现代建筑之间，表现文化被挤压、被遮挡、被局部保留。

## 6. 旁白文案

```text
现代化带来新的居住需求，也带来传统民居的尴尬处境。
当白墙、青瓦、照壁、石墙被水泥盒子替代，失去的不只是建筑样式，也是一套生活组织方式。
```

## 7. 现代文明转折

```text
现代不是问题。
问题是现代建筑是否还能容纳地方记忆。
真正需要设计解决的，不是回到过去，而是在新的居住条件中提炼传统民居的优秀因子。
```

## 8. 对比信息

| 传统民居 | 现代替代 |
|---|---|
| 天井 | 客厅 |
| 照壁 | 外立面 |
| 石墙 | 水泥墙 |
| 院落 | 单元房 |
| 手工装饰 | 工业材料 |
| 家庭礼序 | 标准户型 |
| 自然通风 | 机械设备 |
| 花木庭院 | 封闭阳台 |

## 9. 交互方式

- 拖动滑块切换传统与现代建筑。
- 点击照壁：显示“空间礼序与家庭秩序”。
- 点击石墙：显示“材料记忆与地方性”。
- 点击水泥楼房：显示“效率与同质化”。
- 点击“重新提炼”：出现传统元素转化为现代 UI 卡片的预告。

## 10. 情绪关键词

```text
冲突 / 挤压 / 失落 / 反思 / 转机
```

## 11. 本章短句资产

```text
现代不是问题，失去记忆才是问题。
建筑改变时，生活组织方式也在改变。
传统民居的出路，不是复制过去，而是提炼可以继续生长的因子。
```

# 章节 07｜新的共生：传统成为现代界面

完成叙事收束：传统不再只是被观看的对象，而是成为现代 UI、空间、品牌、建筑、餐饮和公共文化活动的设计语言。

## 2. 章节标题

```text
07 新的共生
传统成为现代界面
```

## 3. 画面构成

画面是一个现代 UI 界面，但界面组件来自传统符号：

## 7. 三类符号的叙事功能总结

### 7.1 身份符号

```text
回答：我们如何被看见？
```

身份符号用于表现身体如何进入族群秩序。  
头饰、耳饰、项圈、冠饰都不是单纯装饰，而是身份的视觉接口。

### 7.2 生活符号

```text
回答：我们如何生活？
```

生活符号用于表现日常生活的器物、动物、植物和劳动方式。  
陶罐、花叶、伞、鸟、犬等符号让用户看到村寨生活、自然环境和家宅经验。

### 7.3 仪式符号

```text
回答：我们如何记住时间、自然与共同体？
```

仪式符号用于表现神圣秩序、祖灵记忆、节庆、舞蹈和共同体关系。  
手掌、兽骨、太阳、双人舞、神鸟等符号让生活进入更高层次的文化记忆。
